# Asteroids
 Clone of Atari Asteroids (1979)
 
 Requirements
 Unity 2021.1.16f1 or higher
 
 Made using Zenject and MVC Pattern
 
 How to Play
 W - Move forward
 A, D - Rotate the Ship
 S - Laser shot
 Space - Gun shot
 
Try to get the highest score as possible!
Good Luck!
