# Asteroids
Clone of Atari Asteroids (1979)
 
## Requirements
Unity 2021.1 or higher
 
Made using Zenject and MVC Pattern
 
## How to Play
* W - Move forward
* A, D - Rotate the Ship
* S - Laser shot
* Space - Gun shot
 
New asteroid will appear after you destroy all of previous ones 
 
Try to get the highest score!
Good Luck!
